# 二、住宿區域與治安原則

## 羅馬

推薦找：

- Roma Termini 西側，往 Repubblica 方向。
- Termini 西南側，靠 Santa Maria Maggiore。
- Via Nazionale 北段。
- 飯店距離車站步行約 5～10 分鐘。

不建議：

- 緊貼 Termini 車站側門。
- 車站東側與東南側過於便宜、偏僻的小巷。
- 需要夜間走地下道才能回飯店的位置。

晚上從 Trastevere 或梵蒂岡回住宿時，優先搭地鐵、計程車或叫車，不必為了省錢從舊城一路走回 Termini。

## 佛羅倫斯

推薦找：

- SMN 車站東側，往舊城區方向。
- Santa Maria Novella 廣場附近。
- 從車站步行 5～10 分鐘、不需穿越偏僻地下道的位置。

佛羅倫斯舊城整體適合步行，但車站周邊仍要注意扒手與主動靠近的人。

## 威尼斯

住宿選擇重點不是景觀，而是：

- 距離 Santa Lucia 車站 5～8 分鐘。
- 最多跨一座橋。
- 有電梯，或房間在低樓層。
- 不要拖行李穿越整個威尼斯。

建議住 Cannaregio 靠車站側。這一區晚上相對有生活感，也比聖馬可廣場附近安靜。

## 米蘭

推薦找：

- Milano Centrale 南側。
- Caiazzo 地鐵站附近。
- Lima／Corso Buenos Aires 靠車站的一端，但不安排逛街。
- 距離機場巴士乘車處約 5～10 分鐘。

避免住在中央車站正門前過於混亂的位置。晚上回飯店不要在車站廣場停留、整理錢包或低頭滑手機。

# 八、交通與防竊原則

## 火車

- 行李盡量放在看得到的位置。
- 車門附近行李架風險較高，若必須放置，可用簡單鋼索鎖固定。
- 高鐵上不要把護照、現金和卡片留在大行李。
- 上下車前先確認月台，不接受陌生人主動帶路或搬行李。

## 市區交通

- 羅馬地鐵與公車、米蘭地鐵、拿坡里 Circumvesuviana 是需要提高警覺的區域。
- 手機不要插在外套或褲子後口袋。
- 背包在人多時移到身前。
- 月台上有人刻意碰撞、遮擋或多人包圍時，立刻移動位置。

## 現金與證件

- 護照、主信用卡與主要現金放貼身防盜包。
- 外層錢包只放當日少量現金與一張備用卡。
- 護照影本、保險資料和訂房資料另存雲端。
- 不在街上拿出整疊現金整理。

## 晚上

- 晚餐後優先搭大眾運輸或計程車回住宿。
- 不為了省 10～20 歐元，深夜走長距離偏僻路段。
- 車站周邊只作為交通轉乘地，不安排深夜散步。

# 十、最終版本的核心取捨

這個版本的重點不是把所有著名景點塞滿，而是：

- 羅馬多住一晚，讓龐貝前後不至於連續爆走。
- 米蘭縮成兩晚，保留大教堂與《最後的晚餐》。
- 完全不安排精品街與百貨購物。
- 威尼斯安排一個完整本島日、一個離島日。
- 佛羅倫斯保留兩個不同類型的一日遊：西恩納，以及比薩＋盧卡。
- 每個城市都有可刪除的備注景點，不必為完成清單而趕路。
- 吃飯每區提供多個選項，現場依排隊人數、休息日和體力決定。
- 最後一天提早前往 BGY，不拿返台航班冒險。
