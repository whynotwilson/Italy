# 一、閱讀方式

## 推薦指數

| 指數 | 定義 |
|---:|---|
| **5** | 核心景點，第一次去最建議保留 |
| **4** | 很推薦，時間與體力允許就去 |
| **3** | 有時間就去，可依現場狀況取捨 |
| **2** | 優先度較低，累了或延誤時可刪 |
| **1** | 最後備選，多餘時間再去 |
| **—** | 交通、用餐、Check-in 等非景點項目 |

## 每日表格格式

| 欄位 | 說明 |
|---|---|
| 時間 | 預計抵達與離開時間 |
| 主行程 | 只寫景點、用餐、移動、Check-in 等主要節點 |
| 可逛多久 | 真正預留給景點的時間 |
| 從上一站的交通 | 交通工具與預估交通時間 |
| 備註 | 安檢、排隊、寄放行李、訂票、體力與路況提醒 |

## 重要觀念

- 表內時間是**規劃目標**，不是已確認的正式班次。
- 高鐵、區域火車、巴士與水上巴士，以 2026 年正式班表和實際購票結果為準。
- 義大利熱門教堂、博物館與古蹟常有安檢，不能只用 Google Maps 的步行時間估算。
- 預約景點建議提早 **20～30 分鐘**到場；大型熱門景點可抓 **30～45 分鐘**。
- 中午抵達新城市時，通常只能寄放行李，**不要預設能提早拿房**。
- 正式 Check-in 通常安排在 15:00 後，或當天觀光結束返回飯店時。
- 若住宿沒有 24 小時櫃台，必須事先確認最晚入住時間。
- 義大利區域火車若持**紙本票**，上車前通常必須在月台附近機器驗票；Trenitalia 電子區域票目前已預先生效，不需另外打票，但仍要依票面規則搭乘。
- 義大利常有交通罷工或臨時施工；城際移動前 48～72 小時要再次查看 Trenitalia、Italo、ACTV、Autolinee Toscane 與機場官方公告。
- 火山與高山行程受天候影響明顯；維蘇威火山及 Dolomites 都不能只看威尼斯／拿坡里市區天氣，必須另查山區能見度、強風、降雪、道路與步道公告。

# 二、五座城市的交通差異

| 城市 | 交通特色 | 容易忽略的地方 |
|---|---|---|
| 羅馬 | 有地鐵 A、B、C 線，另有公車與電車 | 歷史中心很多地方沒有地鐵，最後仍要走不少路；公車可能受塞車影響 |
| 佛羅倫斯 | **沒有地鐵**，市中心主要靠步行、公車與路面電車 | 舊城不大，但石板路多；拖行李時不要只看直線距離 |
| 威尼斯 | 本島沒有汽車與地鐵，主要靠步行與 Vaporetto 水上巴士 | 短短幾百公尺可能要跨好幾座沒有電梯的橋；10 月也要留意 acqua alta 高潮位與臨時改道 |
| 米蘭 | 地鐵系統完整，主要景點多可搭地鐵抵達 | 中央車站周邊方便，但晚上要避免冷清支路 |
| 拿坡里／龐貝 | 拿坡里市區可搭地鐵、步行或計程車；前往龐貝通常從 Napoli Garibaldi 搭 Circumvesuviana | Napoli Centrale 與 Napoli Garibaldi 位於同一交通建築群但樓層、站名不同；車站和列車擁擠時要特別顧好手機與包包 |
| 維蘇威火山 | 從龐貝／拿坡里出發需另接公車、接駁車或套裝交通，火山口入口不在火車站旁 | 必須先買火山口指定時段票；天候警報、強風或步道狀況可能臨時關閉 |
| Dolomites 一日遊 | 從威尼斯搭小團車輛前往東側 Dolomites，通常以 Cortina、Misurina 與沿途觀景點為主 | 不是近郊行程，單日約 9～11 小時；實際停靠點依道路、天氣與季節調整 |

### 是否出發的判斷標準

出發前 48～72 小時查看山區預報，重點不是只看「降雨機率」，而是：

1. Cortina、Misurina 一帶的雲底高度與能見度。
2. 是否有持續降雨、強風、早雪或道路警示。
3. 小團是否確認照常出發，以及是否大幅刪減主要景觀點。
4. 免費取消期限是否已到。

符合下列情況就改用威尼斯備案：

- 山區整日濃霧或低雲，主要山峰大概率看不到。
- 持續中大雨、雷雨、強風或道路可能受積雪影響。
- 業者臨時改成只停 Cortina 市區、幾乎取消湖泊與山景。
- 自己前一天已明顯疲勞，不適合再坐 4～5 小時山路車程。

### 山區衣物與裝備

- 薄羽絨或保暖中層＋防風防水外套。
- 長褲、抓地力好的運動鞋或登山鞋。
- 飲水、能量棒、太陽眼鏡、防曬與帽子。
- 山區氣溫可能比威尼斯低很多，不以威尼斯市區穿著直接出發。
- 不攜帶大行李；只帶小型日用背包。
