# 一、閱讀方式

## 推薦指數

| 指數 | 定義 |
|---:|---|
| **5** | 核心景點，第一次去最建議保留 |
| **4** | 很推薦，時間與體力允許就去 |
| **3** | 有時間就去，可依現場狀況取捨 |
| **2** | 優先度較低，累了或延誤時可刪 |
| **1** | 最後備選，多餘時間再去 |
| **—** | 交通、用餐、Check-in 等非景點項目 |

## 每日表格格式

| 欄位 | 說明 |
|---|---|
| 時間 | 預計抵達與離開時間 |
| 主行程 | 只寫景點、用餐、移動、Check-in 等主要節點 |
| 可逛多久 | 真正預留給景點的時間 |
| 從上一站的交通 | 交通工具與預估交通時間 |
| 備註 | 安檢、排隊、寄放行李、訂票、體力與路況提醒 |

## 重要觀念

- 表內時間是**規劃目標**，不是已確認的正式班次。
- 高鐵、區域火車、巴士與水上巴士，以 2026 年正式班表和實際購票結果為準。
- 義大利熱門教堂、博物館與古蹟常有安檢，不能只用 Google Maps 的步行時間估算。
- 預約景點建議提早 **20～30 分鐘**到場；大型熱門景點可抓 **30～45 分鐘**。
- 中午抵達新城市時，通常只能寄放行李，**不要預設能提早拿房**。
- 正式 Check-in 通常安排在 15:00 後，或當天觀光結束返回飯店時。
- 若住宿沒有 24 小時櫃台，必須事先確認最晚入住時間。
- 義大利區域火車若持**紙本票**，上車前通常必須在月台附近機器驗票；Trenitalia 電子區域票目前已預先生效，不需另外打票，但仍要依票面規則搭乘。
- 義大利常有交通罷工或臨時施工；城際移動前 48～72 小時要再次查看 Trenitalia、Italo、ACTV、Autolinee Toscane 與機場官方公告。

# 二、四座城市的交通差異

| 城市 | 交通特色 | 容易忽略的地方 |
|---|---|---|
| 羅馬 | 有地鐵 A、B、C 線，另有公車與電車 | 歷史中心很多地方沒有地鐵，最後仍要走不少路；公車可能受塞車影響 |
| 佛羅倫斯 | **沒有地鐵**，市中心主要靠步行、公車與路面電車 | 舊城不大，但石板路多；拖行李時不要只看直線距離 |
| 威尼斯 | 本島沒有汽車與地鐵，主要靠步行與 Vaporetto 水上巴士 | 短短幾百公尺可能要跨好幾座沒有電梯的橋；10 月也要留意 acqua alta 高潮位與臨時改道 |
| 米蘭 | 地鐵系統完整，主要景點多可搭地鐵抵達 | 中央車站周邊方便，但晚上要避免冷清支路 |
| 拿坡里／龐貝 | 高鐵到 Napoli Centrale，再轉 Circumvesuviana | Napoli Centrale 與 Napoli Garibaldi 是相連但不同層的車站；轉車要預留時間 |

# 四、這版加入的特色景點

為了避免整趟都是教堂、廣場與純外觀拍照，主行程加入下列較有差異性的景點：

| 城市 | 特色景點 | 特殊之處 |
|---|---|---|
| 羅馬 | 尼祿金宮 Domus Aurea | 地下皇宮遺址，採指定時段參觀，部分票種包含導覽或沉浸式內容 |
| 羅馬 | Centrale Montemartini | 古羅馬雕像展示在舊發電廠的鍋爐與大型機械之間 |
| 佛羅倫斯 | 伽利略博物館 | 科學儀器、天文與伽利略相關收藏 |
| 佛羅倫斯 | 瓦薩里走廊（備選） | 美第奇家族高架通道；特殊但會壓縮烏菲茲參觀時間 |
| 威尼斯 | Burano＋海軍歷史博物館 | 彩色潟湖小島搭配威尼斯造船與海軍歷史，內容變化較大 |
| 威尼斯 | 2026 威尼斯藝術雙年展（備選） | 期間限定，但較適合本來就喜歡當代藝術的人 |
| 米蘭 | 米蘭紀念墓園 | 大型雕塑、家族陵墓與建築構成的戶外藝術空間 |
| 米蘭 | 達文西國立科學技術博物館 | 科學、工業、交通與達文西展區，比純美術館更有變化 |

原本的共和廣場、密涅瓦廣場、法爾內塞廣場、里阿爾托市場街區、Murano 玻璃示範、《最後的晚餐》與部分純外觀景點，改為低優先或備選。這版也避免為了追求「特殊」而把整趟旅行變成連續逛博物館。

# 八、出發前最後確認清單

## 訂票前

- 尼祿金宮必須和羅馬競技場分開購票。
- 瓦薩里走廊只在確定願意縮短烏菲茲時間時才購買。
- 龐貝使用官方指定平台購買記名票，姓名需和證件一致。
- 米蘭大教堂確認票券指定的是教堂、屋頂，還是組合票，以及屋頂的指定入場時間。
- 威尼斯先比較單程票與 ACTV 72 小時票，再決定第一趟何時啟用。

## 出發前 7 天

- 逐家確認餐廳營業日與是否接受單人訂位。
- 確認尼祿金宮、梵蒂岡、烏菲茲、聖馬可、總督宮、米蘭大教堂是否有臨時關閉或宗教活動。
- 查看西恩納與比薩的宗教活動公告。
- 下載離線地圖、所有票券 QR Code 與住宿地址。
- 向四間住宿確認入住前寄放行李與最晚 Check-in 時間。

## 每次城際移動前 48～72 小時

- 查看義大利交通罷工、鐵路施工與班次異動。
- 再確認火車出發車站：Roma Termini、Firenze SMN、Venezia Santa Lucia、Milano Centrale。
- 區域火車紙本票記得驗票；電子票依票面規則使用。
- 西恩納 131R 巴士及威尼斯 Line 12 以當日官方班表重新反推時間。

## 每天早上

- 查看天氣、降雨、體感溫度與日落時間。
- 威尼斯查看 acqua alta 高潮位預報。
- 檢查第一個預約景點的入口名稱，不要只導航到景點中心點。
- 高強度日若腳痛或睡眠不足，優先刪除推薦 3 以下的行程。

1. 避免景點正前方第一排、菜單圖片過多且主動拉客的餐廳。
